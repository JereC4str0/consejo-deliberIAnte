# Changelog

## 2.0.0

- Skill reescrita desde cero con arquitectura más modular y profesional.
- Selector dinámico de roles por señales de dominio, complejidad, urgencia y tono.
- Integración profunda con memoria de Hermes: lectura previa y escritura posterior.
- Sistema de auto-mejora más autónomo, con métricas y proceso de revisión meta.
- Reglas anti-sarasa reforzadas con lista de frases prohibidas y test de calidad final.
- Nuevo modo `--contexto` para forzar lectura de contexto de proyectos activos.
- Proceso deliberativo de 4 fases con rondas de debate estructuradas y puntuación de confianza.
- Plantillas claras de output para `--ultra`, `--quick` y `--deep`.
- Ejemplos ampliados y más detallados.
- Actualizado README y estructura de documentación.

## 1.0.0

- Skill renombrada de `consejo-deliberante-evolutivo` a `consejo-deliberiante`.
- Sistema de triggers ampliado con `deliberia` y `delibera ia`.
- Matriz de selección de roles por tipo de consulta.
- Modos `--ultra`, `--deep`, `--quick`, `--roles`, `--debug`, `--mejora`.
- Reglas anti-sarasa estructurales.
- Proceso deliberativo de 4 fases.
- Integración con memoria de Hermes.
- Sección de auto-mejora.
